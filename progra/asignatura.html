II. Descripción de la asignatura

Los estudiantes comprenden y aplican los lenguajes y tecnologías para el desarrollo de aplicaciones para computadoras, dispositivos móviles; sin embargo, aún no conocen la integración de los elementos de desarrollo de sistemas para la Web que les permita crear soluciones profesionales en entornos distribuidos. Se pretende que, en la asignatura, los estudiantes adquieran las competencias para desarrollar programas orientados a entornos Web.

Con Programación Computacional IV se aplicarán los lenguajes de programación y tecnologías para la Web con el propósito de utilizarlos en el desarrollo de aplicaciones basándose en estándares internacionales de desarrollo de software trabajando de forma individual y colaborativa.

Contenido de la asignatura

UNIDAD I - Desarrollo Frontend con librerías del lado del cliente.

  1. Introducción a la librería Vue.js y Bootstrap
  2. Rendering Declarativo y Manipulación de DOM
  3. Sistema de Componentes

UNIDAD II - Desarrollo Backend con el Framework del lado del servidor y base de datos.

  1. Introducción al desarrollo con PHP
  2. Instalación y configuración del servidor Web
  3. Introducción a Laravel
  4. Conectando a una BD MySQL con Laravel
  5. CRUD con Laravel
  6. Formularios con Laravel
  7. Sistemas de Rutas y controladores
  8. Validación de datos (rutas, vistas, formularios y controladores)
  9. Blade: sistema de plantillas avanzado
  10. Trabajando con Laravel/UI

UNIDAD III - Incorporando tecnologías de Servidor Web.

  1. Introducción a Node.js
  2. Instalación y configuración de Node.js
  3. Asincronismo en Node.js
  4. Módulos Nativos
  5. Introducción a Express
  6. Servicios de datos en Node.js con MongoDB
  7. Aplicaciones Real-Time en Node.js con Socket.io
