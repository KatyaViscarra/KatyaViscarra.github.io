¿Qué es Laravel?

Laravel es un framework para el desarrollo de aplicaciones web con PHP que nos permite construir aplicaciones modernas con una sintaxis elegante y expresiva. Éste fue creado por Taylor Otwell en el 2011 y a la fecha de publicación de este curso va por la versión 5.5. Para empezar a trabajar con Laravel necesitas preparar tu entorno de desarrollo, IDE o editor de texto y otras herramientas como Composer que te ayudarán a instalar y desarrollar aplicaciones web con este framework.

Laravel tiene varias formas de realizar la instalación en un sistema operativo, entre ellos, mediante el uso de Docker Desktop, y la que fue desarrollada durante clases fue mediante la instalación y uso de PHP y Composer, usando el comando “composer global require laravel/installer”.

Nuevo proyecto y vinculación de base de datos

Para crear un nuevo proyecto de Laravel puedes hacerlo de 2 maneras: con el comando create-project de Composer o con el instalador laravel new.

Con ello, para iniciar una conexión a una base de datos seguiremos los siguientes pasos:

  • Crear una base de datos en el gestor de bases de datos que utilizaremos (en este caso, lo haremos en PhpMyAdmin).
  • Identificar y modificar el archivo ".env", para el gestor indicado se debe colocar el puerto correcto, el nombre de la base y las credenciales (usuario y contraseña).
  • Realizar las migraciones de las tablas en Laravel a la base de datos.
